## Micro Journal Rev.6.1: Mini - User Manual

The Micro Journal Rev.6.1: Mini has the same internal hardware as the Micro Journal Rev.6, so you can follow the Rev.6 manual for all general instructions.

The only difference is access to the battery holder. To reach it, you need to remove the four screws on the top cover. The video below shows how to do this:

[Rev.6.1. How to install the battery](https://youtu.be/DYZrfpyJvLY?si=i3qGGrl_JMHrHZpA&t=394)

Please refer to the Rev.6 user manual for all other instructions:

[Rev.6 Quick Start Guide](https://github.com/unkyulee/micro-journal/blob/main/micro-journal-rev-6-one-piece/quickstart.md)
